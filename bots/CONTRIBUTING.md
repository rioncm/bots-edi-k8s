CONTRIBUTING TO BOTS EDI PROJECT
================================

First of all, **thank you** for your interest in contributing to Bots
Open Source EDI translator!

## 1. About the Project

Bots project is licensed under **GPL v3**. That means:
-   Any distributed modifications must also be licensed under GPL v3.
-   Full text of the license and original copyright notices must be always included.

## 2. Getting Started

Our project\'s main repository is maintained at GitLab:
-   source code: <https://gitlab.com/bots-ediint/bots>
-   documentation: <https://gitlab.com/bots-ediint/bots-docs> (Sphinx sources used to
    generate documentation at <https://bots.readthedocs.io>)
-   Bots plugins: <https://gitlab.com/bots-ediint/bots-plugins>
-   Bots grammars: <https://gitlab.com/bots-ediint/bots-grammars>

## 3. How to Contribute

-   Browse open issues labeled [Help wanted].
-   Leave a comment on an issue you intend to work on to avoid
    overlapping efforts.
-   Feel free to contribute to documentation.

## 4. Commit & Code Style

-   Use **clear and descriptive** commit messages (e.g. "Fix smtps
    python 3.13").
-   Follow the existing code style.

## 5. Reporting Bugs & Suggesting Features

-   Use GitLab issues to report bugs or request features.
-   For bugs, include:
    -   A clear title
    -   Steps to reproduce
    -   Expected vs actual behavior
    -   Version and environment details
-   For feature requests, describe:
    -   The problem you\'re solving
    -   Proposed implementation ideas

## 6. Merge Request Process

How to submit your proposed changes:
-   First open an issue to describe the purpose.
-   Then a merge request could be created from this issue with a branch name
    sugested by gitlab starting with the issue number.
-   Pull the repo and checkout to this new branch to work on it.
-   Push your modifications to GitLab.
-   A maintainer will review and may suggest revisions.
-   Contributions can\'t be merged until they comply with GPL v3 terms.

## 7. Other Ways to Help

Not a coder? No problem! You can still contribute by:

-   Improving documentation.
-   Testing new features.
-   Reporting bugs.
-   Sharing your experience with Bots and helping others on our Discord
    group.
-   Spreading the word about Bots.

## 8. License & Code of Conduct

-   This project follows the **GPL v3 license**. Source files must
    comply with license included in each distribution and any
    modifications under the same license.
-   Please review our \[license.rst\] for more details.

## Final Notes

Thanks again for contributing! Your efforts help to build the first Open
Source EDI Translator, available for everyone.
